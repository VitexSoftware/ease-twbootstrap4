Requirements
------------

[vitexsoftware/ease-html](https://github.com/VitexSoftware/ease-html)


Installation
------------

```
composer require vitexsoftware/ease-bootstrap4
```

Also you can use debian package:

```
sudo apt install php-ease-bootstrap4
```
